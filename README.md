# 番茄钟应用 - Pomodoro Timer

一个功能完整、界面简洁美观的番茄钟应用，基于 React + TypeScript 构建，部署在 Cloudflare Pages。

## ✨ 功能特性

### 🍅 番茄钟模式
- 工作时间：25 分钟（可配置）
- 休息时间：5 分钟（可配置）
- 开始/暂停/重置控制
- 时间到达时的声音和桌面通知
- 视觉进度显示

### ⏱️ 正计时功能
- 自定义长度的计时器
- 支持小时、分钟、秒的显示
- 开始/暂停/停止控制
- 实时显示经过时间
- 分段统计显示

### 📅 日历功能
- 月份日历视图
- 展示番茄钟完成记录
- 统计信息展示：
  - 本月总完成数
  - 本周完成数
  - 日均完成数
- 直观的日期标记

## 🎨 设计特点

- 现代化设计风格，界面简洁美观
- 响应式设计，完美支持桌面和移动端
- 深色/浅色主题切换
- 流畅的动画效果
- 直观的用户交互

## 🛠️ 技术栈

- **框架**: React 18
- **语言**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **数据持久化**: localStorage
- **通知**: Web Notifications API

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 📦 部署到 Cloudflare Pages

1. 连接你的 Git 仓库到 Cloudflare Pages
2. 配置构建设置：
   - **构建命令**: `npm run build`
   - **构建输出目录**: `dist`
   - **Node 版本**: 18 或更高

3. 部署即可！

### 本地验证

在部署前，可以运行以下命令验证：

```bash
# 类型检查

# 代码检查
npm run lint

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 📱 使用说明

### 番茄钟
1. 点击"番茄钟"标签
2. 点击"开始"按钮开始工作计时
3. 专注工作 25 分钟
4. 时间到后会收到通知，自动进入 5 分钟休息时间
5. 可通过设置按钮自定义工作和休息时长

### 正计时
1. 点击"正计时"标签
2. 点击"开始"按钮开始计时
3. 适用于任意长度的任务
4. 可随时暂停或停止

### 日历
1. 点击"日历"标签
2. 查看每月的番茄钟完成记录
3. 查看统计数据
4. 每完成一个工作番茄钟，日历会自动记录

## 🌟 特色功能

- **数据持久化**: 所有设置和记录保存在本地，不会丢失
- **桌面通知**: 时间到达时会发送桌面通知（需授权）
- **主题切换**: 支持浅色和深色主题，自动适配系统偏好
- **响应式设计**: 在各种设备上都有出色的体验
