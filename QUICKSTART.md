# 快速开始 - 番茄钟应用

## 📋 项目概述

这是一个功能完整的番茄钟应用，包含三大核心功能：
- 🍅 **番茄钟计时器**：25分钟工作 + 5分钟休息的经典番茄工作法
- ⏱️ **正计时器**：适用于任意长度任务的自定义计时器
- 📅 **日历统计**：可视化展示你的番茄钟完成记录和统计数据

## 🚀 本地运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173 即可使用应用。

## 🔧 可用命令

```bash
npm run dev        # 启动开发服务器
npm run build      # 构建生产版本
npm run preview    # 预览生产构建
npm run typecheck  # TypeScript 类型检查
npm run lint       # ESLint 代码检查
```

## ✨ 主要功能

### 番茄钟模式
- ⏰ 默认 25 分钟工作时间，5 分钟休息时间
- ⚙️ 可自定义工作和休息时长
- 🔔 时间到达时的声音和桌面通知
- 📊 圆形进度条实时显示
- 📝 自动记录完成的番茄钟到日历

### 正计时器
- ⏱️ 无限制的时间累加
- ⏸️ 暂停/继续功能
- 🛑 停止并重置
- 📈 分段显示小时、分钟、秒

### 日历统计
- 📅 月度日历视图
- 🍅 每日番茄钟完成数量标记
- 📊 统计数据：
  - 本月总完成数
  - 本周总完成数  
  - 日均完成数
- 🔄 月份切换导航

## 🎨 主题切换

应用支持浅色和深色两种主题：
- 点击右上角的太阳/月亮图标切换主题
- 主题选择会自动保存到本地
- 首次访问会根据系统偏好自动选择主题

## 💾 数据持久化

所有数据都保存在浏览器的 localStorage 中：
- ⚙️ 番茄钟设置（工作/休息时长）
- 📊 番茄钟完成记录
- 🎨 主题偏好

数据不会丢失，除非手动清除浏览器数据。

## 🔔 桌面通知

首次使用时，应用会请求桌面通知权限：
1. 点击浏览器弹出的权限请求
2. 选择"允许"
3. 当番茄钟时间结束时，你会收到桌面通知

即使标签页在后台，你也能收到通知！

## 📖 使用技巧

1. **专注工作**：使用番茄钟保持 25 分钟的深度专注
2. **定期休息**：不要跳过 5 分钟的休息时间，这对保持长期生产力很重要
3. **记录习惯**：查看日历统计，了解自己的工作习惯
4. **自定义设置**：根据自己的节奏调整工作和休息时长
5. **长任务计时**：使用正计时器记录较长任务的实际耗时

## 🎯 番茄工作法原则

1. **选择任务**：确定要完成的任务
2. **设定番茄钟**：启动 25 分钟计时器
3. **专注工作**：直到番茄钟响起，避免任何干扰
4. **短暂休息**：休息 5 分钟
5. **重复循环**：每完成 4 个番茄钟，休息 15-30 分钟
